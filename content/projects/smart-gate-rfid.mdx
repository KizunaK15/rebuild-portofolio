## Overview

The Smart Gate RFID System automates room access to improve security and throughput. Initial concepts explored visual QR code scanning, but lighting conditions made it unreliable. Moving to an RFID and servo-based architecture provided the stability required for continuous operation.

## Technical Approach

The system uses an Arduino Nano to interface with an RC522 RFID module. When a valid card is detected, the microcontroller sends a precise PWM signal to actuate the servo motor, opening the gate. An isolated power circuit prevents the servo's current draw from dropping the logic voltage.

## Results

The gate authenticates users and opens in under one second. It has successfully processed hundreds of entry events without false rejections or hardware resets.
