## Overview

The Post Stroke Smart Hand Gloves help stroke patients regain hand mobility through guided, data-tracked therapy sessions. Unlike off-the-shelf devices, this system gives therapists real-time visibility into every session — eliminating the need for manual note-taking and enabling remote follow-up.

## Technical Approach

The glove uses flex sensors to detect finger extension angles, which feed into a PID servo controller achieving 95% motion accuracy. An LVGL-driven touchscreen displays the current exercise regime, progress, and session history. Session data is pushed to Firebase Realtime Database in near-real-time; a cloud function triggers a WhatsApp API webhook to notify the assigned therapist after each completed session.

## Results

Awarded Silver Medal at the IIIEX 2025 international invention exhibition in recognition of both the clinical utility and engineering quality of the design. Internal trials showed an 80% reduction in therapist manual reporting time and stable device operation across more than 200 consecutive test sessions without hardware failure.
